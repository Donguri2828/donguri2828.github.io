My Profile

～小学校 幼少の頃よりテレビゲームに熱中。小学校高学年時にPCを買い与えてもらった事をきっかけに、（ゲーム）プログラミングというものに漠然とした興味を持つ。偶然にも高専生だった従兄の話を聞き、高専入学に強く興味を持つ。
中学校 勉強と部活動（剣道部）の日々。部活動では１年の夏に手術を要するほどの大怪我を負ったりと波乱万丈ではあったが、最終的には総体出場と剣道二段を取得。勉強においても第一志望であった茨城高専を受験し無事合格。
高専（本科） 勉強とアルバイトを両立。少々人見知りな性格の自分だが、友人に恵まれた高専生活や、アルバイトを通して、確かな自信が身に付いた。
高専（専攻科） スキルアップと自分への挑戦として、ハッカソン形式のインターンに参加。自分にとって初めての規模のチーム開発であったが、意欲的なチームメイトに恵まれ、チーム開発の楽しさややりがいを知る。
